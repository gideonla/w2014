\subsection*{85}
Suppose that there is only one thread, and it calls \verb|lock| then \verb|unlock|. At this point, the \verb|tail| field of the lock is set to the node of the thread. The next time the thread calls lock, it will receive its own node as \verb|pred|, but it had just set its own node's \verb|locked| field to true, so the while loop will spin forever.
